# 日麻點數計算助手 - 更新日誌

## 🚀 v2.0 - 完整功能版本 (2026-05-10)

### ✨ 新增功能

#### 核心計算引擎
- ✅ **役種判定器** (`yaku_detector.py`)
  - 支持 30+ 役種識別
  - 包括所有役滿判定 (國士無双、大三元、大四喜等)
  - 自動番符計算
  - 符數計算邏輯

- ✅ **手牌分析器** (`hand_analyzer.py`)
  - 勝利手牌判定
  - 手牌結構分析
  - 對子檢測
  - 花色分佈統計

- ✅ **點數計算器** (`points_calculator.py`)
  - 完整的點數對照表
  - 支持親/子家計算
  - 支持自摸/榮和分配
  - 滿貫及以上的特殊計算

#### Web 應用界面
- ✅ **Streamlit Web App** (`app.py`)
  - 響應式移動端設計
  - 友善的用戶界面
  - 實時計算結果展示
  - 詳細的支付信息
  - 手牌結構可視化

#### 測試套件
- ✅ **單元測試** (30+ 測試用例)
  - `test_yaku_detector.py` - 14 個役種判定測試
  - `test_hand_analyzer.py` - 8 個手牌分析測試
  - `test_points_calculator.py` - 10 個點數計算測試

### 🎯 功能特性

#### 支持的役種 (30+)
**1番役**
- 立直、門前清模和、平胡、斷么九、一盃口
- 三色同順、一氣通貫、三色同刻、三槓子、三暗刻
- 小三元、混老頭、混全帶么九

**3-5番役**
- 混一色 (3番) 
- 清一色 (5番)

**役滿 (13番)**
- 國士無双、大三元、大四喜
- 清老頭、字一色、綠一色
- 四暗刻、九蓮寶燈

#### 計分系統
- ✅ 完整的符數對照表
- ✅ 番符轉點數計算
- ✅ 親子家點數差異
- ✅ 自摸榮和支付分配
- ✅ 滿貫、跳滿、倍滿、三倍滿、役滿

## 🚀 快速開始

### 安裝

```bash
pip install -r requirements.txt
```

### 執行應用

```bash
streamlit run app.py
```

應用將在 http://localhost:8501 啟動

### 執行測試

```bash
# 執行所有測試
pytest tests/ -v

# 執行測試並生成覆蓋率報告
pytest tests/ --cov=src --cov-report=html
```

## 💡 使用示例

### 計算平胡立直

1. 輸入手牌: `123m456p789s1122z`
2. 選中: [立直]
3. 結果: 1番30符 → 1000點 (親榮和)

### 計算清一色

1. 輸入手牌: `111m222m333m444m555m66m`
2. 結果: 5番 清一色 → 滿貫 (6000點)

### 計算國士無双

1. 輸入手牌: `19m19p19s1234567z`
2. 結果: 役滿 → 24000點
